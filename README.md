# Aurora Homebrew Studio

A modern desktop application for creating, editing, validating, and exporting custom content for **Aurora Character Builder**.

Aurora Homebrew Studio aims to eliminate manual XML editing by providing a visual interface for creating homebrew content.

---

## Features

### Current

* Modern CustomTkinter interface
* Multi-page application framework

### Planned

* Magic Item Editor
* Equipment Editor
* Spell Editor
* Feat Editor
* Background Editor
* Class Editor
* Subclass Editor
* Species Editor
* Monster Editor
* XML Importer
* XML Exporter
* XML Validator
* Project Save/Load
* Sourcebook Builder
